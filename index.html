Web

Mobile First

There are over 2 billion mobile web users worldwide (Source), more than 50% of African mobile internet users, more than 20% of users in developed markets and 59% of users in India, either never, or infrequently, access the Internet from a computer (Source).

The mobile market is massive, and continues to grow, as such when designing websites I take a mobile first approach, what this means is that when designing the website, I ensure that it looks spectacular on mobile devices first, before adopting it to larger screens, this is called progressive enhancement.

Progressive Enhancement

Graceful Degradation is when you start developing a website adapted to desktop browsers, one issue with this approach is that it takes more effort to remove functionality to adapt a website to mobile devices.

By using the mobile first approach you ensure that your site looks good on mobile devices, you then progressively add features enhancing the website to make the it stand out from the crowd by employing functionality and techniques that would not be possible on mobile devices due to restrictions on current mobile browsers.
